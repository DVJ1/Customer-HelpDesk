# Customer HelpDesk

A MERN-based Helpdesk System for managing customer support tickets with authentication, role-based access, and structured workflow.

## Features
- Ticket creation and management
- Admin assignment system
- Status tracking (Open → In Progress → Resolved)
- Responsive UI

## Tech Stack
- React, Tailwind CSS
- Node.js, Express
- MongoDB

## Setup

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm run dev
